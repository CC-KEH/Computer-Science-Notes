## ğŸ—ï¸ 4. Nested Structures

Objects can contain **arrays**, **other objects**, or both:

```js
let student = {
  name: "John",
  grades: [85, 90, 78],
  address: {
    city: "London",
    zip: "E1 6AN"
  }
};

console.log(student.grades[1]); // 90
console.log(student.address.city); // "London"
```

## ğŸ” 5. Looping Over Objects

### `for...in` loop (for objects):
```js
for (let key in person) {
  console.log(key + ": " + person[key]);
}
```

### `Object.keys()`, `Object.values()`, `Object.entries()`:

```js
console.log(Object.keys(person));   // ["name", "age"]
console.log(Object.values(person)); // ["Alice", 25]
console.log(Object.entries(person)); // [["name", "Alice"], ["age", 25]]
```

## ğŸ”§ 6. Methods in Objects & `this` Keyword

An **object method** is a function stored as a property.

```js
let car = {
  brand: "Toyota",
  drive: function() {
    console.log("Driving a " + this.brand);
  }
};

car.drive(); // "Driving a Toyota"
```

### Shorthand Syntax:
```js
let user = {
  name: "Eve",
  greet() {
    console.log("Hi, " + this.name);
  }
};
```

âœ… `this` refers to the object that calls the method.
