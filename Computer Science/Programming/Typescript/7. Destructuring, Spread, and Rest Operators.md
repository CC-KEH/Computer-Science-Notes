## ðŸŽ’ 3. Rest Syntax (`...`) â€“ The Opposite of Spread

Gathers **remaining values** into an array or object.

---

### Arrays â€“ Collect Remaining Items

```js

```

---

### Objects â€“ Collect Remaining Keys

```js

  name: "Bob",
  age: 30,
  city: "Paris"
};

console.log(others); // { age: 30, city: "Paris" }
```

### Functions â€“ Rest Parameters
